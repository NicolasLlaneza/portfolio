Sobre mí

Desarrollador Full Stack, formado en la Tecnicatura Universitaria en Programación (UTN-FRM). Trabajo cómodo en todo el stack: modelar una base de datos, diseñar una API prolija y construir la interfaz que la consume, sin depender de que otro resuelva las partes que no me tocan.

Ya llevé un proyecto de cliente real a producción —desde el diseño de la base de datos y las políticas de seguridad hasta el deploy— y en paralelo vengo construyendo proyectos propios (una arquitectura de integración con IA, una app de fútbol, un e-commerce en equipo) para seguir ampliando el stack con el que trabajo. Lo que más me atrae es el lado backend: modelar bien un dominio, pensar los casos borde, y dejar una base de datos que cualquier otro developer pueda entender.

JavaScript / TypeScript React Node.js / Express Python / FastAPI PostgreSQL Supabase n8n Docker Java
